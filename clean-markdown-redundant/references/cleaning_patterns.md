## OCR 图片注释块

### 标准格式

OCR 图片注释块的典型格式:

```html
<!-- 这是一张图片，ocr内容为：图片中的文字说明 -->
```

### 变体形式

脚本需要识别并处理以下所有变体:

#### 1. 标点符号变体

```html
<!-- 这是一张图片,ocr内容为:说明 -->
<!-- 这是一张图片，OCR内容为：说明 -->
<!-- 这是一张图片,OCR 内容为:说明 -->
```

**处理**: 忽略中英文标点符号的差异(逗号、冒号的中英文形式)。

#### 2. 空格变体

```html
<!--这是一张图片，ocr内容为：说明-->
<!-- 这是一张图片，ocr内容为：说明 -->
<!--  这是一张图片，  ocr内容为：说明  -->
```

**处理**: 注释标记内部可能有多余空格,仍需匹配。

#### 3. 大小写变体

```html
<!-- 这是一张图片，OCR内容为：说明 -->
<!-- 这是一张图片，Ocr内容为：说明 -->
<!-- 这是一张图片，ocr内容为：说明 -->
```

**处理**: 忽略大小写差异。

#### 4. 多行变体

```html
<!-- 这是一张图片，ocr内容为：
这是一个很长的
多行说明文字
-->
```

**处理**: 支持多行匹配,删除完整的注释块。

### 正则表达式

```python
pattern = r'<!--\s*这是一张图片[,，]?\s*ocr\s*内容为\s*[:：].*?-->'
```

**说明**:
- `<!--\s*` - 注释开始标记,后跟任意空格
- `这是一张图片` - 固定文本
- `[,，]?` - 可选的逗号(中英文)
- `\s*` - 任意空格
- `ocr` - 固定文本(忽略大小写)
- `\s*内容为\s*` - 固定文本,前后可能有空格
- `[:：]` - 英文或中文冒号
- `.*?` - 任意内容(非贪婪匹配)
- `-->` - 注释结束标记

**标志**:
- `re.DOTALL` - 让 `.` 匹配换行符
- `re.IGNORECASE` - 忽略大小写

## 字体颜色标签

### 标准格式

字体颜色标签的典型格式:

```html
<font style="color:rgba(255,0,0,1);">红色文字</font>
```

### 变体形式

#### 1. 空格变体

```html
<font style="color: rgba(255, 0, 0, 1);">红色文字</font>
<font style="color:rgba(255,0,0,1)">红色文字</font>
<font style = "color:rgba(255,0,0,1);">红色文字</font>
```

**处理**: 属性名、等号、冒号、括号前后都可能有空格。

#### 2. 分号变体

```html
<font style="color:rgba(255,0,0,1);">红色文字</font>
<font style="color:rgba(255,0,0,1)">红色文字</font>
```

**处理**: 分号可选。

#### 3. 引号变体

```html
<font style="color:rgba(255,0,0,1);">红色文字</font>
<font style='color:rgba(255,0,0,1);'>红色文字</font>
```

**处理**: 支持单引号和双引号。

#### 4. Alpha 值变体

```html
<font style="color:rgba(255,0,0,1);">不透明</font>
<font style="color:rgba(255,0,0,0.5);">半透明</font>
<font style="color:rgba(255,0,0,0.123);">自定义透明度</font>
```

**处理**: Alpha 值可以是整数或小数。

### 正则表达式

**开头标签**:

```python
opening_pattern = (
    r'<font\s+'  # 标签名和至少一个空格
    r'style\s*=\s*'  # style属性
    r'["\']'  # 引号开始
    r'color\s*:\s*'  # color属性
    r'rgba\s*\('  # rgba函数
    r'\s*\d+\s*,\s*\d+\s*,\s*\d+\s*,\s*[\d.]+'  # 四个参数(数字,数字,数字,数字或小数)
    r'\s*\)'  # 右括号
    r'\s*;?'  # 可选分号
    r'\s*["\']'  # 引号结束
    r'\s*>'  # 标签结束
)
```

**结尾标签**:

```python
closing_pattern = r'</font>'
```

**处理逻辑**:
1. 先删除所有匹配的开头标签
2. 再删除所有的结尾标签
3. 保留标签内的文字内容

## 边界情况处理

### 1. 嵌套标签

**情况**:

```html
<font style="color:rgba(255,0,0,1);">外层<font style="color:rgba(0,0,255,1);">内层</font>文字</font>
```

**处理**: 逐个删除所有标签,结果为:

```text
外层内层文字
```

**注意**: 不推荐使用嵌套标签,但如果出现,脚本仍能正确处理。

### 2. 跨行标签

**情况**:

```html
<font style="color:rgba(255,0,0,1);">
跨多行的
文字内容
</font>
```

**处理**: 删除标签,保留文字:

```text
跨多行的
文字内容
```

### 3. OCR 注释在行内

**情况**:

```markdown
这是一段文字<!-- 这是一张图片，ocr内容为：说明 -->继续的文字
```

**处理**: 删除注释后,前后文字自然衔接:

```markdown
这是一段文字继续的文字
```

### 4. 标签内的特殊字符

**情况**:

```html
<font style="color:rgba(255,0,0,1);">包含 <特殊> & 字符的文本</font>
```

**处理**: 保留标签内的所有内容:

```text
包含 <特殊> & 字符的文本
```

### 5. 多个 OCR 注释连续出现

**情况**:

```markdown
<!-- 这是一张图片，ocr内容为：图片1 -->
<!-- 这是一张图片，ocr内容为：图片2 -->
<!-- 这是一张图片，ocr内容为：图片3 -->
```

**处理**: 删除所有注释块,并清理多余空行:

```markdown

```

(结果为空,或仅保留一个空行)

### 6. OCR 注释中包含 HTML 标签

**情况**:

```html
<!-- 这是一张图片，ocr内容为：<div>复杂的HTML内容</div> -->
```

**处理**: 整个注释块被完整删除,包括内部的 HTML。

## 保留的内容

以下内容**不删除**:

### 1. 普通 HTML 注释

```html
<!-- 这是普通的注释,不是 OCR 注释 -->
<!-- TODO: 需要修改的内容 -->
<!-- NOTE: 注意事项 -->
```

**判断依据**: 只有包含 "这是一张图片" 和 "ocr内容为" 的注释才会被删除。

### 2. 其他 HTML 标签

```html
<div>容器</div>
<span>行内元素</span>
<br>
<hr>
<a href="url">链接</a>
```

**判断依据**: 只删除 `<font style="color:rgba(...)">` 标签,其他标签保留。

### 3. Markdown 格式

```markdown
# 标题
- 列表项
**粗体** *斜体*
[链接](url)
```

**处理**: 所有 Markdown 格式保持不变。

### 4. 其他 font 标签

```html
<font face="宋体">指定字体</font>
<font size="4">指定大小</font>
<font color="red">指定颜色名</font>
```

**判断依据**: 只删除带有 `style="color:rgba(...)"` 的 font 标签,其他 font 标签保留。

## 清理后的处理

### 1. 空行清理

删除冗余内容后,可能会产生多余的空行:

```markdown
段落1

<!-- 注释 -->

段落2
```

**处理**: 将连续的 3 个或更多空行压缩为 2 个空行:

```markdown
段落1

段落2
```

### 2. 文件末尾清理

确保文件以单个换行符结尾,没有多余的空行。

### 3. 空格清理

如果标签被删除后,在文字之间留下多余空格:

```markdown
文字<font>内容</font>结束
```

**处理**: 当前脚本会保留标签内的文字,结果为:

```markdown
文字内容结束
```

**注意**: 如果需要删除文字间的多余空格,可以添加额外的清理步骤。
